# Payments Manager

## Setup

1. Install Node.js (v18+).
2. Open a terminal in this directory (`payments-manager`).
3. Run `npm install`.
4. Create a Supabase project at https://supabase.com.
5. Run the SQL from `supabase/schema.sql` in your Supabase SQL Editor to set up the database Tables and Policies.
6. Create a `.env` file based on `.env.example` (`cp .env.example .env`) and add your Supabase URL and Anon Key.
7. Run `npm run dev` to start the app.
